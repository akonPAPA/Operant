# Core API

Java 21 Spring Boot 3.x application. The core API owns the future business mutation path for OrderPilot.

## Run locally

```powershell
cd "C:\OrderPilot\OrderPilot-Core\apps\core-api"
mvn spring-boot:run
```

## Test

```powershell
cd "C:\OrderPilot\OrderPilot-Core\apps\core-api"
mvn test
```

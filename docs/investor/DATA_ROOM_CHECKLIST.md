# Investor Data Room Checklist

## Product

- Product one-pager
- Core v1 scope
- Roadmap
- Demo script
- Screenshots or demo video
- Security memo
- Architecture overview

## Market

- ICP definition
- Pain statement
- Competitor comparison
- TAM/SAM/SOM model
- Pricing logic
- ROI model

## Traction

- Design partner list
- Interview notes
- LOIs or pilot interest
- Demo feedback
- Usage metrics if available

## Technical

- Architecture docs
- Security principles
- Data authority model
- AI governance model
- Integration strategy
- Test summary
- Known limitations

## Business

- Founder story
- Hiring plan
- Use of funds
- Financial model
- Pilot offer
- Go-to-market plan

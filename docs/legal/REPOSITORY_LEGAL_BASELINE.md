# Repository Legal Baseline Checklist

This checklist tracks the proprietary legal/IP baseline for the OrderPilot repository.
It is an operational aid, not formal legal advice.

- [x] LICENSE exists and is proprietary (`LICENSE`).
- [x] NOTICE exists (`NOTICE`).
- [x] THIRD_PARTY_NOTICES.md exists (`THIRD_PARTY_NOTICES.md`).
- [x] README has a proprietary notice (`README.md`).
- [x] package.json files are private and UNLICENSED where applicable
      (`apps/web-dashboard/package.json`).
- [x] pyproject.toml / pom.xml metadata is proprietary where safe
      (`apps/ai-worker/pyproject.toml`, `apps/core-api/pom.xml`).
- [x] Open-source dependency policy exists (`docs/legal/OPEN_SOURCE_POLICY.md`).
- [x] AI code usage policy exists (`docs/legal/AI_CODE_USAGE_POLICY.md`).
- [x] IP policy exists (`docs/legal/IP_POLICY.md`).
- [x] SECURITY.md exists (`SECURITY.md`).
- [x] No GPL/AGPL dependency is intentionally introduced by this task.
- [x] No secrets added.
- [x] No product code refactor performed.

# Source Header Template

Optional copyright/license headers for OrderPilot source files. These are templates
only. Do not mass-add headers to every source file as part of a single change — that
creates excessive diff noise. Apply incrementally where useful.

The SPDX identifier `LicenseRef-OrderPilot-Proprietary` denotes the proprietary
OrderPilot license (see `LICENSE`); it is not an OSI/SPDX public license.

## Java / TypeScript

```java
// Copyright (c) 2026 OrderPilot / [Legal Entity Name]. All rights reserved.
// SPDX-License-Identifier: LicenseRef-OrderPilot-Proprietary
```

```typescript
// Copyright (c) 2026 OrderPilot / [Legal Entity Name]. All rights reserved.
// SPDX-License-Identifier: LicenseRef-OrderPilot-Proprietary
```

## Python

```python
# Copyright (c) 2026 OrderPilot / [Legal Entity Name]. All rights reserved.
# SPDX-License-Identifier: LicenseRef-OrderPilot-Proprietary
```
